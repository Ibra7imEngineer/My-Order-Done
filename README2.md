# README2 — اللغات المستخدمة في المشروع ووظائفها

هذا الملف يحدد اللغات والتقنيات التي استُخدمت في مشروع **My Order** وما هي وظيفة كل واحدة وأين توجد ملفاتها الرئيسية.

---

## اللغات والتقنيات المستخدمة

### PHP

- **الاستخدام**: منطق الخادم (Backend)، نقاط الـ API، جلسات المستخدم، عمليات قاعدة البيانات
- **أمثلة ملفات**: `api/*.php`, `admin/*.php`, `index.php`

### HTML

- **الاستخدام**: هيكل صفحات الواجهة الأمامية (Templates، صفحات ثابتة)
- **أمثلة ملفات**: `index.html`

### CSS

- **الاستخدام**: أنماط التصميم، ثيمات فاتح/داكن، استجابة الأجهزة
- **أمثلة ملفات**: `style.css`

### JavaScript

- **الاستخدام**: تفاعل الواجهة، منطق العميل (السلة، البحث، واجهة المستخدم)، استدعاءات AJAX/Fetch إلى الـ API، Service Worker
- **أمثلة ملفات**: `script.js`, `register-sw.js`, `sw.js`, `db-seed.js`

### SQL (MySQL / MariaDB)

- **الاستخدام**: تصميم واستعلام قاعدة البيانات، جداول المنتجات والطلبات والمستخدمين
- **أمثلة ملفات**: `schema.sql`

### JSON

- **الاستخدام**: ملفات إعدادات وتكوين (Manifest، Settings)، تبادل بيانات بين الواجهة والخادم
- **أمثلة ملفات**: `manifest.json`, `settings.json`

### Shell / Batch / PowerShell

- **الاستخدام**: سكربتات بدء الخادم محليًا
- **أمثلة ملفات**: `START_SERVER.bat`, `START_SERVER.ps1`, `START_HERE.sh`

### Markdown

- **الاستخدام**: توثيق المشروع داخليًا
- **أمثلة ملفات**: `README.md`, `README2.md`

## خريطة ملفات مفصّلة حسب اللغة

### PHP

| الملف                    | الوظيفة                                         |
| ------------------------ | ----------------------------------------------- |
| `index.php`              | نقطة الدخول/الواجهة الديناميكية في الجذر        |
| `admin/index.php`        | الصفحة الرئيسية للوحة الإدارة                   |
| `admin/login.php`        | صفحة تسجيل الدخول للإدارة                       |
| `admin/dashboard.php`    | لوحة تحكم الإدارة الرئيسية                      |
| `admin/logout.php`       | تسجيل الخروج من لوحة الإدارة                    |
| `api/config.php.example` | إعدادات اتصال قاعدة البيانات (نسخ وتعديل القيم) |
| `api/db.php`             | دوال اتصال بقاعدة البيانات وتسهيل الاستعلامات   |
| `api/order.php`          | API معالجة طلب واحد                             |
| `api/orders.php`         | API جلب جميع الطلبات                            |
| `api/update_order.php`   | API تحديث حالة الطلب                            |
| `api/stats.php`          | API إحصائيات الطلبات                            |
| `api/log.php`            | API سجل العمليات                                |

### HTML

| الملف        | الوظيفة                          |
| ------------ | -------------------------------- |
| `index.html` | الصفحة الرئيسية للواجهة الأمامية |

### CSS

| الملف       | الوظيفة                                      |
| ----------- | -------------------------------------------- |
| `style.css` | أنماط الموقع العامة، ثيم اللون، تجاوب الشاشة |

### JavaScript

| الملف            | الوظيفة                                              |
| ---------------- | ---------------------------------------------------- |
| `script.js`      | منطق الواجهة الأمامية (السلة، التفاعلات، fetch/ajax) |
| `db-seed.js`     | سكربت Node.js لملء بيانات تجريبية                    |
| `register-sw.js` | تسجيل Service Worker                                 |
| `sw.js`          | Service Worker ومنطق الكاش / العمل بدون اتصال        |

### SQL

| الملف        | الوظيفة                                           |
| ------------ | ------------------------------------------------- |
| `schema.sql` | مخطط قاعدة البيانات، يحتوي تعريف الجداول الأساسية |

### JSON

| الملف           | الوظيفة                         |
| --------------- | ------------------------------- |
| `manifest.json` | تعريف PWA، أيقونات واسم التطبيق |
| `settings.json` | إعدادات التطبيق العامة          |

### Shell / Batch / PowerShell

| الملف              | الوظيفة                      |
| ------------------ | ---------------------------- |
| `START_SERVER.bat` | تشغيل الخادم على Windows     |
| `START_SERVER.ps1` | تشغيل الخادم عبر PowerShell  |
| `START_HERE.sh`    | أوامر بدء سريعة على Unix/Mac |

### Icons / Assets

| الملف                         | الوظيفة              |
| ----------------------------- | -------------------- |
| `icons/generate-icons.html`   | أداة توليد الأيقونات |
| `icons/icon-192.svg`          | أيقونة 192x192       |
| `icons/icon-512.svg`          | أيقونة 512x512       |
| `icons/safari-pinned-tab.svg` | أيقونة Safari        |

### ملفات نصية ومساعدة

| الملف        | الوظيفة                      |
| ------------ | ---------------------------- |
| `الرابط.txt` | رابط النشر أو معلومات الموقع |

## إرشادات للتطوير المستقبلي

- **إضافة لغة جديدة**: أنشئ مجلدًا منفصلاً وحدّد عملية البناء
- **سكربتات التهيئة**: ضعها في مجلد `scripts/` ووثّقها في README
- **تبعيات npm**: أضف `package.json` إذا احتجت لمكتبات Node.js
- **Firebase**: أضف `firebase-config.js` إذا رغبت في التكامل
