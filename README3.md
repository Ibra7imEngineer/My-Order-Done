# README3 — دليل اللغات والتقنيات المستخدمة (نسخة عربية كاملة)

هذا الملف يقدّم وصفاً مفصّلاً لكل لغة أو تقنية مستخدمة في مشروع My Order، أمثلة على الأماكن التي وُظفت فيها، وإرشادات سريعة لتعديل أو تشغيل الأجزاء المرتبطة.

## هدف الملف

يوفّر هذا الدليل مرجعًا سريعًا لأي مطوّر جديد يريد فهم بنية المشروع وأين يتم استخدام كل تقنية، بالإضافة إلى تعليمات تشغيل وتحرير أساسية.

---

## نظرة عامة على اللغات والتقنيات

- **HTML**
  - الوظيفة: بنية الصفحات والتيمبلتات، محتوى الواجهة الثابت.
  - أمثلة ونماذج: `index.html`, `icons/generate-icons.html`, صفحات داخل مجلد `admin/`.
  - ملاحظات: عدّل النصوص الظاهرة والوسوم<meta> هنا لتغيير العنوان والوصف العام للموقع.

- **CSS**
  - الوظيفة: أنماط التصميم، الألوان، التخطيط والاستجابة (responsive).
  - أمثلة: `style.css` و‑أنماط مضمنة داخل بعض ملفات HTML وملفات لوحة الإدارة.
  - ملاحظات: ألوان الثيم الرئيسية موجودة كمتغيرات CSS في أعلى `style.css`.

- **JavaScript**
  - الوظيفة: تفاعل المستخدم، إدارة السلة، منطق الطلب، توليد الفواتير، تكامل PWA وService Worker.
  - ملفات رئيسية: `script.js` (المنطق الرئيسي)، `register-sw.js` (تسجيل الـ SW)، `sw.js` (service worker)، `db-seed.js` (أدوات اختبار/تعبئة بيانات).
  - ملاحظات: لمعظم النصوص الديناميكية ستجد السلاسل داخل `script.js` التي يجب تعديلها للترجمة أو تغيير رسائل المستخدم.

- **PHP**
  - الوظيفة: منطق الخادم (Backend)، نقاط API، إدارة الجلسات وصفحات لوحة الإدارة.
  - أمثلة: مجلد `api/` (`api/order.php`, `api/orders.php`, `api/db.php`, إلخ)، مجلد `admin/` (`index.php`, `dashboard.php`, `login.php`, `logout.php`).
  - ملاحظات: إعدادات الاتصال بقاعدة البيانات في `api/db.php` وملف `api/config.php` إن وجد.

- **SQL**
  - الوظيفة: تعريف الجداول، المفاتيح، والفهارس، وبناء قاعدة البيانات التشغيلية.
  - أمثلة: `schema.sql` يحتوي على جداول `orders`, `activity_logs`, `user_sessions`, إلخ.
  - ملاحظات: نفّذ `schema.sql` في MySQL/MariaDB لإنشاء الجداول المطلوبة.

- **JSON**
  - الوظيفة: إعدادات التطبيق وبيانات الـ PWA.
  - أمثلة: `manifest.json` (تعريف التطبيق كتطبيق ويب تقدّمي)، `settings.json` (إعدادات محلية إن وُجدت).

- **Shell / Batch / PowerShell**
  - الوظيفة: سكربتات مساعدة لتشغيل أو إعداد بيئة التطوير محلياً.
  - أمثلة: `START_HERE.sh`, `START_SERVER.bat`, `START_SERVER.ps1`.
  - ملاحظات: على Windows استخدم `START_SERVER.bat` أو `START_SERVER.ps1` لتشغيل سيرفر بسيط، أو ضع المشروع داخل مجلد XAMPP ثم شغّل Apache.

- **أصول ورسوميات / أيقونات**
  - المجلد: `icons/` يحتوي على أيقونات التطبيق وملفات `apple-touch-icon` وما إلى ذلك.

- **وثائق وملاحق**
  - أمثلة: `README.md`, `README2.md`, `RUN_LOCALLY.md`, `README3.md` (هذا الملف).

## أمثلة عملية سريعة

- لتغيير اسم الموقع الظاهر في الواجهة: عدّل العنوان ووسوم الميتا واسم الشعار في `index.html` و`manifest.json` وداخل `script.js` حيث توجد النصوص.
- لتعديل إعدادات الاتصال بقاعدة البيانات: افتح `api/db.php` وعدّل بيانات المضيف، المستخدم، وكلمة المرور.
- لتغيير شكل الأيقونات: ضع صور جديدة في `icons/` واستبدل المراجع في `manifest.json` وملفات `index.html` و`icons/generate-icons.html`.

## تشغيل المشروع محلياً (ملاحظات)

1. عبر XAMPP (موصى به لبيئة PHP/MySQL):
   - ضع المجلد `My-Order-main` داخل `c:/xampp/htdocs/`.
   - شغّل Apache وMySQL من لوحة XAMPP.
   - افتح المتصفح: `http://localhost/My-Order-main/` أو `http://localhost/My-Order-main/admin/dashboard.php`.

2. بديل: خادم ثابت (لأغراض تصفح الواجهة فقط):
   - باستخدام Python في مجلد المشروع:

```powershell
python -m http.server 8000
```

- افتح: `http://localhost:8000/` (لن تعمل وظائف PHP/MySQL بهذه الطريقة).

## كيفية المساهمة أو تعديل السلاسل النصية

- النصوص الظاهرة للمستخدم (الرسائل، العناوين، إشعارات الواتساب/الرسائل) غالبًا موجودة داخل `script.js` وملفات `admin/*.php`.
- قبل تغيير الكلمات الثابتة راجع ما إذا كانت مستخدمة أيضًا في خوادم الرسائل أو تكامل الطرف الثالث (مثل Firebase أو WhatsApp templates).

---

## ملفات مهمة لاستعراض سريع

- `index.html` — الصفحة الرئيسية والواجهة الأمامية
- `script.js` — منطق الواجهة والسلوكيات الرئيسية
- `style.css` — أنماط وتصميم الواجهة
- `manifest.json` — إعدادات PWA
- `sw.js` — Service Worker (التخزين المؤقت / offline)
- `api/` — نقاط النهاية للخادم (PHP)
- `schema.sql` — تعريف قاعدة البيانات
